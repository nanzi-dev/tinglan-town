# 《听澜镇》视觉验收

视觉验收覆盖 `1280×720` 桌面视口和 `960×540` 小窗口视口。截图由真实
Godot 主场景生成，包含标题、常驻 HUD、公告板、居民对话、关系档案和暂停
界面。

## 生成截图

```bash
.tools/godot/godot --path . --resolution 1280x720 \
  --script res://tests/visual/capture_visuals.gd -- --size 1280x720

.tools/godot/godot --path . --resolution 960x540 \
  --script res://tests/visual/capture_visuals.gd -- --size 960x540
```

截图写入 `tests/visual/screenshots/`。每个视口应生成：

- `title`：标题文字直接覆盖江南春季关键画面，无标题卡片。
- `gameplay`：常驻 HUD 不遮挡主要世界视野。
- `task-board`：公告板表单与操作按钮完整可见。
- `conversation`：参与者头像、对话正文和输入操作不重叠。
- `relationships`：居民头像、公开关系阶段与近期原因清晰可读。
- `pause`：暂停背景、状态文字和继续按钮完整可见。

## 人工检查

- 关键画面无水印、错误文字或主体裁切。
- 头像图集包含十名成人，排列为五列两行，无水印和错误文字。
- 纸纹理低对比，不降低正文可读性。
- 两个视口均无 UI 重叠、中文溢出或横向裁切。
- 标题、公告板、对话、关系档案和暂停界面均有清晰键盘焦点。
- `Esc` 先关闭当前覆盖面板，再切换暂停；暂停状态下再次按下可继续游戏。

## 2026-07-17 验收结果

- 已生成并检查两个视口的全部 12 张截图，文件尺寸与命名正确。
- `960×540` 公告板切换为纵向紧凑布局，表单字段保持对齐且无横向裁切；
  其余字段可通过表单内的纵向滚动访问。
- `1280×720` 公告板保留双栏布局，表单和本次发布列表边界清晰。
- 暂停关键画面使用完全不透明的暗化层，不与实时 3D 世界叠加。
- 标题、游戏 HUD、对话和关系档案在两个视口均无文字或控件重叠。
